## ArtLibro 소개 
<br>
최근, 정보화 사회를 살아가며 넘쳐나는 데이터와 볼거리 속 문화/예술은 과거보다 우리에게 더 가깝게 다가오고 있습니다. 
도서 , 도서관에 대한 정보를 제공하며 또한 공연, 행사, 연극 등의 문화/예술과 도서를 간편하고 상세하게 찾아보고 추천받을 수 있는 
문화 플랫폼입니다.
<br>
<br>
<br>
